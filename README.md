## Run Locally

Clone the project

```bash
  git clone https://github.com/thatsfov1/meetfriends.git
```

Go to the project directory

```bash
  cd meetfriends-main
```

Install dependencies

```bash
  npm install
```



## Running Tests

To run tests, run the following command

```bash
  npm run test
```
